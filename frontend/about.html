SetupRoaster

When AI was still a phenomenon, a friend and I decided that eventually, we would make a PC setup roaster. This year, we decided it was time.

How was it made?

SetupRoaster was created in Node.js (backend) and vanilla JavaScript (frontend). The frontend sends an image to the backend, which sends it to Google's Gemini API. Then, it takes the JSON response, converts it to plain text, and sends it back to the frontend for you to see.
